# 需求

1. 页面展示商品列表（价格，名称，库存。。。）
2. 列表上面有一个搜索栏，可以进行模糊查询。查询之后只显示匹配的商品
3. 每个商品后面有一个购买按钮。
4. 每人有一个钱包，每次购买完后会扣钱（钱显示在页面上）。买完之后减库存
